## 보호, 보안의 원칙과 목적에 대해 이해한다. (일부)

- 보호 : 내부적 문제 사전적 보호
  - (OS내에서 할 수 있는 일) : 계정분리, 리소스 퍼미션
- 보안 : 보호 + 외부적 문제 사전적/사후적 보호
  - 암호화, 인증, 방화벽 등

## 추가 키워드 :
